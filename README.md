# Sweet Mae's Cookie Co. Website

This repository contains the source code for the Sweet Mae's Cookie Co. website. This is a static website that tells the story of the company and showcases its products.

The user has expressed an interest in redeveloping this website under a new project name, "bake". This repository serves as a backup of the original "cookie" project.

## Website Content

The website consists of several HTML pages:

-   `story.html`: The main page, which tells the story of the company's heritage.
-   `faqs.html`: Frequently asked questions.
-   `find.html`: Information on where to find the cookies.
-   `ep-173-master-p-reviews-unboxing-sweet-maes-cookies.html`: A blog post or review.
-   `give-the-gift-of-sweetness-from-black-bakers-and-winemakers-essence-magazine.html`: A blog post or feature.

The website has a "Order A Batch" call to action that links to an external website (`smcookieco.wpengine.com`).

## Technical Details

The website is built with the following technologies:

-   **HTML:** The core language for the website's structure.
-   **CSS:** For styling the website. It uses several CSS files, including a main stylesheet (`sweet-maes-style.css`) and styles from WordPress plugins.
-   **JavaScript:** For interactivity. It uses jQuery and other JavaScript libraries.
-   **Fonts:** It uses external fonts from Typekit and also has local font files.

The website appears to be a static version of a WordPress site, as evidenced by the file structure and references to `wp-content` and `wp-json`.

## Assets

The website's assets are organized into the following folders:

-   `css/`: Contains all the CSS files.
-   `favicons/`: Contains the website's favicons.
-   `fonts/`: Contains local font files and icons.
-   `images/`: Contains all the images used on the website, including logos, patterns, and illustrations.
-   `js/`: Contains all the JavaScript files.
